### 环境准备

安装以下内容

-   [git](https://git-scm.com/)
-   [node](https://nodejs.org/en/)

### 克隆仓库

直接克隆本仓库或者 fork

```shell
git clone https://github.com/StreakingMan/solvable-sheep-game.git
```

### 依赖安装

克隆到本地后，在项目内敲命令

```shell
npm install -g yarn

yarn install
```

### 调试运行

```shell
yarn dev
```
